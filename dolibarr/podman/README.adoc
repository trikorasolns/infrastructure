== Install

Create the POD.

[source,bash]
----
podman pod create --label app=dolibarr \
  --label app.kubernetes.io/name="dolibarr" \
  --label app.kubernetes.io/instance="dolibarr" \
  --label app.kubernetes.io/version="23" \
  --label app.kubernetes.io/component="application" \
  --label app.kubernetes.io/part-of="dolibarr" \
  -p 7280:80 -p 7206:3306 \
  dolibarr 
----

Deploy the MariaDB database.

[source,bash]
----
ansible-playbook mariadb/podman/ansible/podman-install-playbook.yaml \
  -e @dolibarr/podman/ansible/defaults/main.yaml \
  -e db_password="${DB_PASSWORD:-dolibarr}"
----

[source,bash]
----
ansible-playbook dolibarr/podman/ansible/dolibarr-install-playbook.yaml \
  -e @dolibarr/podman/ansible/defaults/main.yaml \
  -e doli_company_name=<company name>
----


== Uninstall

Uninstall Dolibarr.

[source,bash]
----
ansible-playbook dolibarr/podman/ansible/dolibarr-uninstall-playbook.yaml \
  -e @dolibarr/podman/ansible/defaults/main.yaml
----

Uninstall the PostgreSQL database.

[source,bash]
----
ansible-playbook mariadb/podman/ansible/podman-uninstall-playbook.yaml \
  -e @dolibarr/podman/ansible/defaults/main.yaml
----

== Upgrade

Uninstall the Dolibarr pod.

Redeploy the Dolibarr Pod using a new version, _e.g._ `-e dolibarr_version="22.0.4"`.

Open the Dolibarr URL and follow the on screen instructions to update the database.
