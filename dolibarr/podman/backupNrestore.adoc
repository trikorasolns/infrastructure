== Restore

=== Database Restore

==== PostgreSQL Restore

Copy the database backup to the PostgreSQL POD.

[source,bash]
----
podman cp /z/_backup/apps/dolibarr/20260205T133756.dolibarr.backup.sql dolibarr_pgsql_dolibarr:/tmp/
----  

Open a bash shell on the Podman PostgreSQL container.

[source,bash]
----
podman exec -it dolibarr_pgsql_dolibarr bash
----

Restore the database.

[source,bash]
----
psql  -d dolibarr -U dolibarr -f /tmp/20260205T133756.dolibarr.backup.sql
----
