== Preparation

Define backup environment variables.

Set the root location of the backup folder.

[source,bash]
----
export BACKUP_ROOT_FOLDER=/z/_backup/apps
----

Set the kube config file.

[source,bash]
----  
export KUBECONFIG=~/.kube/config
----
